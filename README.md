As funcionalidades da API permitem: criar novos usuários, buscar usuários por ID, atualizar informações existentes e remover usuários do banco de dados.

Todo o fluxo foi testado utilizando o Postman, garantindo o correto funcionamento das rotas e das operações no banco de dados.
